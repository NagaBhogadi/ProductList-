# Product List App

Product List App is an iOS application built using Swift and UIKit.  
The app displays a list of products using UITableView.

## Features

- Displays product list in a UITableView
- Shows product name, id, and description
- Uses custom Product model
- Supports mock data
- Uses protocol for dependency injection
- Supports NetworkManager and MockNetworkManager
- Built using MVC architecture

## Technologies Used

- Swift
- UIKit
- Xcode
- UITableView
- Protocol
- Dependency Injection
- Singleton
- MVC Architecture
